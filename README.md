# MiniProject2

Link to Website: https://jstark220.github.io/MiniProject2/ 

What the tool does: This tool is a net worth tracker; it allows for users to input and save their financial data and visualize the allocation of their net worth. It also uses industry resources to reccomend allocation percentages. 

How to use it: To use the app, you simply use the dropdown to select the type of asset you want to record, label the asset how you want, and set the amount. Once you are done, scroll down and use the visualization function to visualize your data in different ways. You can save your net worth as a snapshot and compare it to your breakdown in future time periods on the saved snapshots page. The recomendations page will make recomendations based on your inputs. 

What I learned: I learned the importance of proper function and javascript use. I had no idea how to save data locally, and make it intuitive so that the tool would actually be useful. I also learned that more inportant that pure functionality, is the ease of use for the user. A lot of the preliminary versions that AI helped me with were not intuitive, and sometimes I did not even know how to use the functions. I tinkered with this a lot so that the app is easily interpretted by anyone.
