The first piece of javascript that I focused on was the dropdown menu. This was the perfect opportunity to use an array of options. I began without giving users the option to add their own type of assets, so I opted for a dropdown menu with an array of options (Stocks, Real Estate, etc). There is also a form function that users use for both the label of the asset (ticket, type of property, etc), and for the value of the asset. There is no choice for visualization yet, but the form entry renders in a list below. I started with only one page of HTML that is pretty basic. There are two sections; the first is a basic setup with divs that contains the form for users to enter their assets. The second is set up with a table since it displays the user's entry with value. Eventually, there will be more pages for users to visualize their data and track their performance.
